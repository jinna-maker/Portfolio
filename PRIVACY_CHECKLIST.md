# Privacy Checklist for Child Portfolio

Before making anything public, check this list.

## Do not publish

- Home address
- Phone number
- Full birthdate
- School ID
- Passport or government ID
- Private email
- Raw certificate folders
- Private Google Drive folders
- Daily school schedule
- Unreviewed family photos
- Photos showing private location details
- Teacher/judge feedback that was not meant to be public

## Safe to publish

- First name or full name, depending on purpose
- Grade level
- School name, if appropriate
- Project name
- Project year
- Public project summary
- Selected project photo
- Demo video without private location details
- Public report/poster
- Public GitHub code repository
- Confirmed award names

## Website publishing rule

Only rows with:

```text
Publish = Yes
Visibility = Public
```

should appear on the public website.

## Google Sheet warning

If a tab is published to web as CSV, anyone with the CSV link may be able to access that published data.
Only publish public-safe tabs and public-safe columns.
